# Manoj Kumar Sekar - Digital Analyst Portfolio

## Setup Instructions

This project was built with React + Vite.

### Prerequisites

- Node.js (v16 or higher)

### Installation

1. Open your terminal in this folder.
2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server:

```bash
npm run dev
```

Then open `http://localhost:5173` in your browser.

## Features

- **Dark Data Theme**: Deep navy background with Neon Cyan/Purple accents.
- **Interactive Graphs**: Radar charts for skills using `recharts`.
- **Animated Background**: Framer Motion particles.
- **Responsive**: Fully optimized for Desktop and Mobile.
